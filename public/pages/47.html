What is Javascript Object :

A javaScript object is an entity having state and behavior (properties and method). For example: car, pen, bike, chair, glass, keyboard, monitor etc. JavaScript is an object-based language. Everything is an object in JavaScript..

Syntax of Javascript Object :


var person =  {f_name:"jitendra", l_name:"sahu", age:"23"};



//print the object

console.log(person)



//print the specific object element

console.log(person.f_name);

console.log(person.l_name)

console.log(person.age)





//define array of object

var persons = [ {f_name:"jitendra", l_name:"sahu", age:"23"} , {f_name:"jitendra", l_name:"sahu", age:"23"} ];



//print the array of object

console.log(persons);



//print the single object from array 

console.log(persons[0]);



//print the element of object from array of object

console.log(persons[0].l_name);
